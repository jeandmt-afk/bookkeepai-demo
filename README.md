# 💰 BookkeepAI Demo

A smart bookkeeping web app designed to help users manage transactions, track expenses, and simulate real-world financial workflows with AI-assisted categorization.

---

## 🚀 Features

* 📊 Transaction tracking (Personal, Business, Utilities)
* 🤖 AI-based categorization system
* 📁 CSV import & export
* 🧾 Bills manager + Financial assistant
* 🧠 AI-powered financial insights
* 🏗️ Project-based bookkeeping workspace

---

## 🖥️ How to Run Locally

### 1. Install dependencies

```bash
pip install streamlit pandas
```

### 2. Run the app

```bash
streamlit run My_Books.py
```
